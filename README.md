# rawr - Discohook Embed Generator

A simple selfbot with a cute anime GUI that converts raw Discord message JSON to Discohook-compatible embeds.

## Features:
- Paste raw Discord JSON or upload a `.json` file.
- Generate Discohook-compatible embeds with optional customizations.
- Save the generated embed to a `.json` file.
- Open Discohook directly with pre-filled embed data.
